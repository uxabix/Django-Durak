# 🃏 Django Durak Card Game

An online multiplayer version of the classic Russian card game **Durak**, built with **Django** and **Django Channels**.  

---

## 🚀 Stack
- Django, Channels  
- PostgreSQL  
- Docker  
- GitFlow  
- Sphinx
